YouTube Realtime

YouTube Realtime is a simple experiment which shows the current live subscriber count on YouTube for a user. The count displayed on YouTube is often incorrect because it doesn't update in realtime. The count on this page is taken directly from the API and hence is guaranteed to be accurate.

Go try it: YouTube Realtime

Did I mention this is an open-source app? You can fork it and make it suite your needs, optionally submit pull requests to make it better. Get it: github.com/akshatmittal/youtube-realtime
